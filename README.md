## 🚀 Overview

Welcome to **SynAura**, CRTX’s primary framework.

SynAura is a **local-first, centralized, multi-functional environment** designed to unify tools, workflows, hobbies, and system-level interaction into a single intelligent interface.

Built with a **polyglot, modular, and constantly evolving architecture**, SynAura is not static—it adapts, improves, and expands over time.

> The system you always wished existed.

---

## 🧠 AI System

SynAura integrates local AI using **Ollama**, enabling:

- Fully **local LLM execution**
- **Interchangeable models** (swap depending on task)
- No forced reliance on external APIs
- A foundation for **intelligent system control**

The AI is not just for chat.

It is being designed to:
- Explain code  
- Execute actions  
- Navigate systems  
- Integrate directly into workflows
