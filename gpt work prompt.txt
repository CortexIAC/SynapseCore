# 🧠 PROJECT OVERVIEW

AI OS is a **modular, multi-interface, AI-assisted desktop environment** built in Python (Tkinter currently), designed to function like a lightweight operating system workspace.

It includes:

* Dynamic UI
* Multiple “Work Interfaces”
* Voice control (hold ALT to speak)
* AI chat + future code assistant
* Modular tab system (Notebook-based)
* Runtime-controlled architecture

---

# 🧱 CURRENT ARCHITECTURE

## Core System

* runtime.py → controls app lifecycle, routing, logging, voice start
* state.py → shared system state
* voice_controller.py → handles ALT-hold voice input loop
* settings system (planned but not fully implemented)

## UI Layer

* main_window.py → PRIMARY UI CONTROLLER (DO NOT BREAK)
* startup.py → intro screen
* main_menu.py → launcher

## Tabs / Modules

Each tab is modular and attached to ttk.Notebook:

* ChatTab → AI interaction (supports "chat" and "code" mode)
* DebugTab → logs (connected via runtime.debug_callback)
* FileManagerTab → file navigation
* SystemMonitorTab → system stats
* ControlPanel → admin tools
